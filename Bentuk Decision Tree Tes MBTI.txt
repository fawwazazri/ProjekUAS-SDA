## 📊 Breakdown Per Level

### **LEVEL 1: Ekstrovert vs Introvert (E/I)**
```
ROOT: "Apakah Anda lebih berenergi saat berinteraksi dengan banyak orang?"
├── TIDAK → Introvert (I)
└── YA → Extrovert (E)
```

### **LEVEL 2: Sensing vs Intuition (S/N)**
```
Introvert Branch:
├── Introvert + Sensing (IS_)
│   └── "Apakah Anda lebih fokus pada fakta dan detail konkret?"
└── Introvert + Intuition (IN_)
    └── "Apakah Anda lebih suka ide-ide baru dan kemungkinan masa depan?"

Extrovert Branch:
├── Extrovert + Sensing (ES_)
│   └── "Apakah Anda lebih fokus pada fakta dan detail konkret?"
└── Extrovert + Intuition (EN_)
    └── "Apakah Anda lebih suka ide-ide baru dan kemungkinan masa depan?"
```

### **LEVEL 3: Thinking vs Feeling (T/F)**
```
IS Branch:
├── IST_ → "Apakah Anda membuat keputusan berdasarkan logika objektif?"
└── ISF_ → "Apakah Anda lebih mengutamakan harmoni dan perasaan orang lain?"

IN Branch:
├── INT_ → "Apakah Anda cenderung kritis dan analitis dalam mengambil keputusan?"
└── INF_ → "Apakah Anda lebih mempertimbangkan dampak emosional dari keputusan?"

ES Branch:
├── EST_ → "Apakah Anda membuat keputusan berdasarkan logika objektif?"
└── ESF_ → "Apakah Anda lebih mengutamakan harmoni dan perasaan orang lain?"

EN Branch:
├── ENT_ → "Apakah Anda cenderung kritis dan analitis dalam mengambil keputusan?"
└── ENF_ → "Apakah Anda lebih mempertimbangkan dampak emosional dari keputusan?"
```

### **LEVEL 4: Judging vs Perceiving (J/P) - LEAF NODES**
```
ISTJ ← IST + YA (Judging)      |  ISTP ← IST + TIDAK (Perceiving)
ISFJ ← ISF + YA (Judging)      |  ISFP ← ISF + TIDAK (Perceiving)
INTJ ← INT + YA (Judging)      |  INTP ← INT + TIDAK (Perceiving)
INFJ ← INF + YA (Judging)      |  INFP ← INF + TIDAK (Perceiving)
ESTJ ← EST + YA (Judging)      |  ESTP ← EST + TIDAK (Perceiving)
ESFJ ← ESF + YA (Judging)      |  ESFP ← ESF + TIDAK (Perceiving)
ENTJ ← ENT + YA (Judging)      |  ENTP ← ENT + TIDAK (Perceiving)
ENFJ ← ENF + YA (Judging)      |  ENFP ← ENF + TIDAK (Perceiving)
```
